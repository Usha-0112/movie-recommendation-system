## Movie Recommendation System :
This project is a movie recommendation web application that suggests movies based on user preferences.  
It uses Machine Learning techniques to recommend similar movies based on content similarity.
Developed by: Ushasri Potluri
Technologies used: Python, Pandas, Scikit-learn, Streamlit
## Features :
Movie similarity recommendation
Interactive web interface
Fast search suggestions
## Technologies Used :
Python
Machine Learning
Pandas
Streamlit
